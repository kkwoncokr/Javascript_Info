## 주석

### 알게된 점

1. 주석(Comment)은 어떻게 코드가 작동하는지, 왜 코드가 동작하는지 설명하는데 쓰인다.
2. 좋은 코드엔 설명이 담긴 주석이 많아선 안된다. 주석없이 코드 자체만으로 코드가 무슨일을 하는지 이해해야함
3. 주석내용으로 설명을 써놓는것도 좋지만, 함수를 생성하여 하는것이 좋은 코드다.

```javascript
// i가 소수인지를 확인함
for (let j = 2; j < i; j++) {
  if (i % j == 0) continue nextPrime;
}
```

```javascript
function isPrime(n) {
  for (let i = 2; i < n; i++) {
    if (n % i == 0) return false;
  }
```

### 느낀점

1. 코드를 짤때 정말 다른 사람이 봐도 알아볼수 있을 정도로 간편하게 짜야한다는 것을 느낀다.
2. 꼭 지켜야 할 규칙은 없지만, 사람들이 자주 사용하는 것을 따라하는것도 나쁘지 않은 듯하다.
