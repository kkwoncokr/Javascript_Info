### while과 for 반복문

1. while 반복문

```javascript
let i = 0;
while (i < 3) {
	// 0, 1, 2가 출력됩니다.
	alert(i);
	i++;
}
```

- i가 0이 되면 반복문을 멈춥니다. falsy로 바뀌기 때문

```javascript
let i = 0;
do {
	alert(i);
	i++;
} while (i < 3);
```

1. do안에 있는 본문은 조건이 truthy, flasy든 상관없이 최소 한번 실행된다.

```javascript
let i = 0;
for (let i = 0; i < 3; i++) {
	// 0, 1, 2가 출력됩니다.
	alert(i);
}
```

1.i가 2가 될때까지 반복한다. 2. 반복문을 빠져 나올땐 break 3. 현재 반복을 종료하고 다음 반복으로 넘어가고 싶은땐 continue
